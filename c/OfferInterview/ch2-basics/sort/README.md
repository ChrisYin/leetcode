# Sorting Algorithms

## Simple Sorting Algorithms

+ 平均inversion的个数: N(N-1)/2
+ 简单排序算法(交换相邻元素)平均需要O(N^2)的时间

## Insertion Sort

+ 时间复杂度:
	+ 平均时间复杂度: O(N^2)
	+ 最差时间复杂度: O(N^2)
	+ 最优时间复杂度: O(N)
+ 无额外空间消耗

## Shell Sort

### Shell's increments

+ 最差时间复杂度: O(N^2)

## Heap Sort

+ 平均比较次数: 2NlogN - O(N)

## Merge Sort

+ 最差时间复杂度: O(NlogN)

## Quick Sort

+ 平均时间复杂度: O(NlogN)
+ 最优时间复杂度: O(NlogN)
+ 最差时间复杂度: O(N^2)
